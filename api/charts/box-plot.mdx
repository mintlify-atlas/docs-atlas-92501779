# BoxPlot

The `BoxPlot` component visualizes statistical distributions using box-and-whisker diagrams, showing the median, quartiles, and outliers. It's ideal for comparing distributions across categories.

## Props

### Data Props

<ParamField path="data" type="Query | array" required>
  Query result or array of objects containing the data to visualize
</ParamField>

<ParamField path="name" type="string" required>
  Column name for category labels (x-axis)
</ParamField>

<ParamField path="min" type="string" required>
  Column name containing minimum values (lower whisker)
</ParamField>

<ParamField path="intervalBottom" type="string" required>
  Column name containing lower quartile (Q1) values
</ParamField>

<ParamField path="midpoint" type="string" required>
  Column name containing median values
</ParamField>

<ParamField path="intervalTop" type="string" required>
  Column name containing upper quartile (Q3) values
</ParamField>

<ParamField path="max" type="string" required>
  Column name containing maximum values (upper whisker)
</ParamField>

<ParamField path="confidenceInterval" type="string">
  Column name for confidence interval data
</ParamField>

## Examples

### Basic Box Plot

```markdown
```sql response_time_stats
select 
  endpoint,
  min(response_time) as min_time,
  percentile_cont(0.25) within group (order by response_time) as q1,
  median(response_time) as median_time,
  percentile_cont(0.75) within group (order by response_time) as q3,
  max(response_time) as max_time
from api_logs
group by endpoint
```

<BoxPlot 
  data={response_time_stats} 
  name="endpoint"
  min="min_time"
  intervalBottom="q1"
  midpoint="median_time"
  intervalTop="q3"
  max="max_time"
/>
```

### Horizontal Box Plot

```markdown
<BoxPlot 
  data={response_time_stats} 
  name="endpoint"
  min="min_time"
  intervalBottom="q1"
  midpoint="median_time"
  intervalTop="q3"
  max="max_time"
  swapXY="true"
  title="API Response Times by Endpoint"
/>
```

### Customized Box Plot

```markdown
<BoxPlot 
  data={response_time_stats} 
  name="endpoint"
  min="min_time"
  intervalBottom="q1"
  midpoint="median_time"
  intervalTop="q3"
  max="max_time"
  color="primary"
  yFmt="#,##0"
  title="Response Time Distribution"
  yAxisTitle="Time (ms)"
  yGridlines="true"
/>
```

### Box Plot with Custom Height

```markdown
```sql salary_by_department
select 
  department,
  min(salary) as min_sal,
  percentile_cont(0.25) within group (order by salary) as q1_sal,
  median(salary) as median_sal,
  percentile_cont(0.75) within group (order by salary) as q3_sal,
  max(salary) as max_sal
from employees
group by department
```

<BoxPlot 
  data={salary_by_department} 
  name="department"
  min="min_sal"
  intervalBottom="q1_sal"
  midpoint="median_sal"
  intervalTop="q3_sal"
  max="max_sal"
  chartAreaHeight="400"
  yFmt="usd0"
  title="Salary Distribution by Department"
/>
```
