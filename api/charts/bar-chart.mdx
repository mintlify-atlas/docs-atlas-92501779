# BarChart

The `BarChart` component visualizes data using rectangular bars. It supports multiple series with stacked, grouped, or 100% stacked layouts, and can be displayed vertically or horizontally.

## Props

### Data Props

<ParamField path="data" type="Query | array" required>
  Query result or array of objects containing the data to visualize
</ParamField>

<ParamField path="x" type="string" required>
  Column name for the x-axis (categories)
</ParamField>

<ParamField path="y" type="string" required>
  Column name for the y-axis (values)
</ParamField>

<ParamField path="series" type="string">
  Column name to group data into multiple series
</ParamField>

<ParamField path="y2" type="string">
  Column name for secondary y-axis values
</ParamField>

### Bar Style Props

<ParamField path="type" type="string" default="stacked">
  Bar layout type: 'stacked' | 'grouped' | 'stacked100'
</ParamField>

<ParamField path="fillColor" type="string">
  Fill color for bars (hex, rgb, or theme color name)
</ParamField>

<ParamField path="fillOpacity" type="number">
  Opacity of bar fill (0-1)
</ParamField>

<ParamField path="outlineColor" type="string">
  Color for bar outlines
</ParamField>

<ParamField path="outlineWidth" type="number">
  Width of bar outlines in pixels
</ParamField>

### Layout Props

<ParamField path="swapXY" type="boolean" default={false}>
  Swap axes to create horizontal bars
</ParamField>

<ParamField path="showAllXAxisLabels" type="boolean">
  Force showing all x-axis labels (may overlap)
</ParamField>

### Formatting Props

<ParamField path="xFmt" type="string">
  Format string for x-axis values
</ParamField>

<ParamField path="yFmt" type="string">
  Format string for y-axis values
</ParamField>

<ParamField path="y2Fmt" type="string">
  Format string for secondary y-axis values
</ParamField>

### Chart Layout Props

<ParamField path="title" type="string">
  Chart title displayed at the top
</ParamField>

<ParamField path="subtitle" type="string">
  Chart subtitle displayed below the title
</ParamField>

<ParamField path="legend" type="boolean">
  Whether to show the legend
</ParamField>

<ParamField path="chartAreaHeight" type="number">
  Height of the chart area in pixels
</ParamField>

<ParamField path="yBaseline" type="boolean">

<ParamField path="y2Baseline" type="boolean">

<ParamField path="xTickMarks" type="boolean">

</ParamField>

<ParamField path="yTickMarks" type="boolean">

</ParamField>

<ParamField path="y2TickMarks" type="boolean">

## Examples

### Basic Bar Chart

```markdown
```sql sales_by_category
select category, sum(sales) as total_sales
from orders
group by category
order by total_sales desc
```

<BarChart data={sales_by_category} x="category" y="total_sales" />
```

### Horizontal Bar Chart

```markdown
<BarChart 
  data={sales_by_category} 
  x="category" 
  y="total_sales" 
  swapXY="true"
  title="Sales by Category"
  yFmt="usd"
/>
```

### Stacked Bar Chart

```markdown
```sql sales_by_month_category
select 
  date_trunc('month', order_date) as month,
  category,
  sum(sales) as total_sales
from orders
group by 1, 2
```

<BarChart 
  data={sales_by_month_category} 
  x="month" 
  y="total_sales" 
  series="category"
  type="stacked"
/>
```

### Grouped Bar Chart

```markdown
<BarChart 
  data={sales_by_month_category} 
  x="month" 
  y="total_sales" 
  series="category"
  type="grouped"
  title="Monthly Sales Comparison"
/>
```

### 100% Stacked Bar Chart

```markdown
<BarChart 
  data={sales_by_month_category} 
  x="month" 
  y="total_sales" 
  series="category"
  type="stacked100"
  title="Sales Distribution"
  yFmt="pct"
/>
```

### Custom Colors and Labels

```markdown
<BarChart 
  data={sales_by_category} 
  x="category" 
  y="total_sales"
  fillColor="primary"
  labels="true"
  labelFmt="usd0"
  yFmt="usd"
  yGridlines="true"
/>
```
