# AreaChart

The `AreaChart` component visualizes data as filled areas, making it ideal for showing cumulative values, trends, and proportions. It supports stacked and 100% stacked layouts with optional line overlays.

## Props

### Data Props

<ParamField path="data" type="Query | array" required>
  Query result or array of objects containing the data to visualize
</ParamField>

<ParamField path="x" type="string" required>
  Column name for the x-axis values
</ParamField>

<ParamField path="y" type="string" required>
  Column name for the y-axis values
</ParamField>

<ParamField path="series" type="string">
  Column name to group data into multiple series
</ParamField>

### Area Style Props

<ParamField path="type" type="string" default="stacked">
  Area layout type: 'stacked' | 'stacked100'
</ParamField>

<ParamField path="fillColor" type="string">
  Fill color for the area (hex, rgb, or theme color name)
</ParamField>

<ParamField path="fillOpacity" type="number">
  Opacity of the area fill (0-1)
</ParamField>

<ParamField path="line" type="boolean">
  Show line at the top edge of the area
</ParamField>

<ParamField path="lineColor" type="string">
  Color for the line overlay
</ParamField>

### Formatting Props

<ParamField path="xFmt" type="string">
  Format string for x-axis values
</ParamField>

<ParamField path="yFmt" type="string">
  Format string for y-axis values
</ParamField>

### Chart Layout Props

<ParamField path="title" type="string">
  Chart title displayed at the top
</ParamField>

<ParamField path="subtitle" type="string">
  Chart subtitle displayed below the title
</ParamField>

<ParamField path="legend" type="boolean">
  Whether to show the legend
</ParamField>

<ParamField path="chartAreaHeight" type="number">
  Height of the chart area in pixels
</ParamField>

<ParamField path="yAxisTitle" type="string | boolean">

<ParamField path="xGridlines" type="boolean">

<ParamField path="xAxisLabels" type="boolean">

</ParamField>

<ParamField path="yAxisLabels" type="boolean">

</ParamField>

<ParamField path="xBaseline" type="boolean">

</ParamField>

<ParamField path="yBaseline" type="boolean">

</ParamField>

<ParamField path="xTickMarks" type="boolean">
  Show tick marks on x-axis
</ParamField>

<ParamField path="yTickMarks" type="boolean">
  Show tick marks on y-axis
</ParamField>

### Axis Scale Props

</ParamField>

<ParamField path="yLogBase" type="number">
  Base for logarithmic y-axis
</ParamField>

<ParamField path="yMin" type="number">
  Minimum value for y-axis
</ParamField>

<ParamField path="yMax" type="number">
  Maximum value for y-axis

<ParamField path="yScale" type="boolean">
  Auto-scale y-axis to data range

<ParamField path="markers" type="boolean">
  Show markers at data points
</ParamField>

<ParamField path="markerShape" type="string">
  Shape of markers: 'circle' | 'square' | 'triangle' | 'diamond'

<ParamField path="markerSize" type="number">
  Size of markers in pixels
</ParamField>

### Line Behavior Props

<ParamField path="step" type="boolean">

<ParamField path="stepPosition" type="string">

## Examples

### Basic Area Chart

```markdown
```sql revenue_over_time
select date_trunc('month', order_date) as month, sum(sales) as revenue
from orders
group by 1
order by 1
```

<AreaChart data={revenue_over_time} x="month" y="revenue" />
```

### Stacked Area Chart

```markdown
```sql revenue_by_category
select 
  date_trunc('month', order_date) as month,
  category,
  sum(sales) as revenue
from orders
group by 1, 2
```

<AreaChart 
  data={revenue_by_category} 
  x="month" 
  y="revenue" 
  series="category"
  title="Revenue by Category Over Time"
  legend="true"
/>
```

### 100% Stacked Area Chart

```markdown
<AreaChart 
  data={revenue_by_category} 
  x="month" 
  y="revenue" 
  series="category"
  type="stacked100"
  title="Revenue Distribution"
  yFmt="pct"
/>
```

### Area Chart with Line Overlay

```markdown
<AreaChart 
  data={revenue_over_time} 
  x="month" 
  y="revenue"
  line="true"
  fillOpacity=0.3
  yFmt="usd"
/>
```

### Custom Colors and Styling

```markdown
<AreaChart 
  data={revenue_by_category} 
  x="month" 
  y="revenue" 
  series="category"
  colorPalette={['#4F46E5', '#7C3AED', '#DB2777']}
  fillOpacity=0.6
  markers="true"
  yGridlines="true"
/>
```

### Step Area Chart

```markdown
<AreaChart 
  data={inventory_levels} 
  x="date" 
  y="quantity"
  step="true"
  stepPosition="end"
  title="Inventory Levels"
/>
```
